# 野球肩コンディション入力

野球肩の状態、整形外科テスト、肩甲骨評価、体幹機能を記録する簡易Webアプリです。

## 公開方法

GitHub Pagesでは `docs/index.html` を公開元に設定します。

1. GitHubで新規リポジトリを作成
2. このフォルダをpush
3. GitHubの `Settings` → `Pages`
4. `Build and deployment` を `Deploy from a branch`
5. Branchを `main`、Folderを `/docs` に設定

入力データはブラウザのローカル保存に記録され、サーバーには送信されません。
